# auto_linux

Installers for üíßDrupal and üçäOrangeHRM.

#### Make the installers:
``` bash
make
```

#### Install Drupal:
``` bash
./drupal_auto_linux:
  --dbuser="drupal"
  --dbname="drupal"
  --dbhost="localhost"
  --dbport="3306"
  --admin="admin"
  --admin_pass="ubuntu"
  --site_mail="noone#nowhere.no"
  --site_name="name"
```

#### Install OrangeHRM:
``` bash
./ohrm_auto_linux:
  --dbuser="ohrmuser"
  --dbname="ohrm"
  --dbhost="localhost"
  --dbport="3306"
  --admin="admin"
  --admin_pass="ubuntu"
```
